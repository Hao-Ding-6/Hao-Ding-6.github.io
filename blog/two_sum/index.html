1. Two Sum

Nice two sum problem

December 18, 2018 Hao Ding

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

Description

Given an array of integers and a value, determine if there are any two integers in the array which sum equal to the given value.

For example:

Given nums = [2, 7, 11, 15], target = 9, Because nums[0] + nums[1] = 2 + 7 = 9, return [0, 1].

Solution

The most brute-force solution is to iterate the array using two pointers i, for each position of i, we iterate the part after i of that array using pointer j, if nums[i] + nums[j] == target, we return the result [i, j]. The time complexity is O(n2).

We can optimize the algorithm to linear time complexity using HashMap. We use a dictionary as a hash table, we use only one pointer i to iterate the array and store the num - index pair to the dictionary. For each position, we check if target - num is in the hash table, if in, we return the result [d[target - num], i]; else, we go to the next position. Here the time complexity is O(n), the memory complexity is O(n).

Here is the Python 3 code.

class Solution:
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        d = collections.defaultdict(int)
        for i, num in enumerate(nums):
            if target - num in d:
                return [d[target - num], i]
            d[num] = i
        return []
