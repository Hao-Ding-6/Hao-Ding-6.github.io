2. Add Two Numbers

Add two numbers problem

December 19, 2018 Hao Ding

2 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

Description

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8
Explanation: 342 + 465 = 807.

Solution

Just iterate two lists:

  • For each position, check if pointer l1 or pointer l2 is NULL, if it is NULL, the value of that pointer is 0; else the value is l1.val and l2.val.
  • Then Sum = val1 + val2 + carry, the variable called carry means the carry digit, initialized as 0.
  • Calculate the value of current position Sum % 10 and next carry Sum // 10.
  • If the pointer l1 or l2 is NULL, stay where it is; else, move to next position.
  • return the final result.

Here is the python 3 code.

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def addTwoNumbers(self, l1, l2):
        """
        :type l1: ListNode
        :type l2: ListNode
        :rtype: ListNode
        """
        carry = 0
        dummy = p = ListNode(-1)
        while l1 or l2:
            val1 = l1.val if l1 else 0
            val2 = l2.val if l2 else 0
            Sum = val1 + val2 + carry
            carry = Sum// 10
            p.next = ListNode(Sum % 10)
            p = p.next
            l1 = l1.next if l1 else l1
            l2 = l2.next if l2 else l2
            
        if carry:
            p.next = ListNode(carry)
        return dummy.next
